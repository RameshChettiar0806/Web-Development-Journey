### What are ID's and Classes

## ID
It also stands for unique identifier assigned to only a particular HTML element in the same webpage

You can make changed to only that HTML Element

In CSS you denote ID's with #. Eg: #red
## Class
It lets you give the same name for multiple HTML Elements.

You can make changes to multiple different HTML Elements at the same time 

You can denote classes with '.'. Eg: .animal
