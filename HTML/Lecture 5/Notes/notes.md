## Table Tags
![Tables](/screenshots/table.png)
-They are used to include tables or represent data in the form of tables in a webpage <table>
- caption tag is used for table name
- always work with  tr tag that is table row
- inside each tr work with td tag that is table description/table column

## Lists
![Lists](/screenshots/Lists.png)
-There are multiple types of lists:
- Ordered Lists (ol tag)
- Unordered Lists(ul tag)

-Under each list you can include li tag or list item which is basically the content of each point of a list( li tag)
